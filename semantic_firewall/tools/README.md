# Scripts

Developer/helper scripts live here.

- `evaluate.py`: standalone detector evaluation report generator.
- `check_db.py`: quick audit DB inspection.
- `debug.py`: ad-hoc detector debugging helper.
- `advanced_features_demo.py`: manual demo for allowlist and multi-turn behavior.
- `neuralchemy_dataset_smoke.py`: quick dataset download and sample firewall check.
- `smoke_test.ps1`: PowerShell smoke test for imports, API routes, and optional pytest.

Run these tools directly from `semantic_firewall/tools/`; the repo no longer keeps root-level wrapper scripts for them.
